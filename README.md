# celebrity-hologram

## Local setup
create-react-app for client
localhost:3000
```sh

$ git clone https://github.com/andyliu-417/demo.git

$ cd demo/client

$ npm install

$ npm start

```


express-generator for server 
localhost:4000
```sh

$ git clone https://github.com/andyliu-417/demo.git

$ cd demo/server

$ npm install

$ npm start

```
